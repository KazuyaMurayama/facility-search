## 🏠 そもそもエアコン掃除ってなぜ必要？（3行で）

エアコンの中はカビ・ホコリ・菌が繁殖しやすい場所。  
そのまま使うと、汚れた空気を部屋中に吹き出し続けることになる。  
プロに内部を高圧洗浄（＝強い水で汚れを吹き飛ばす方法）してもらうことで、電気代の節約にもなる。

## 🚀 申し込みステップ（3ステップ）

### 共通の準備：事前に確認すること

#### STEP 0（準備）: エアコンの情報を手元に用意する

```
確認場所：室内機（壁にかかっているエアコン）の側面か底面にシールが貼ってある

確認項目：
  ① メーカー名（例: ダイキン、パナソニック、三菱など）
  ② 型番（例: AN22YCS-W）
  ③ 製造年（例: 2019年製）
  ④ 「お掃除機能付き」かどうか（型番で検索するとわかる）
  ⑤ 台数
  ⑥ 設置場所の特殊事情（高所・吹き抜けなど）
```

> ⚠️ **「お掃除機能付き」かどうかで料金が約2倍変わる！** 通常: 約7,000〜10,000円 vs お掃除機能付き: 約15,000〜20,000円

## ⚠️ 悪徳業者に騙されないための「赤信号チェックリスト」

これに当てはまる業者は**絶対に使ってはいけない**！

- [ ] 🚨 料金が**5,000円以下**（激安には理由がある→当日追加請求の罠）
- [ ] 🚨 **書面・メールで見積もりを出さない**（口頭だけ）
- [ ] 🚨 **会社の住所・固定電話が不明**（サイトに情報なし）
- [ ] 🚨 **損害賠償保険に入っていない**（作業中の故障・水漏れは自腹になる）
- [ ] 🚨 口コミが**全部★5で文章が不自然**（サクラの可能性）
- [ ] 🚨 **強引な追加オプション営業**（「防カビは必須です！」など）
- [ ] 🚨 **作業時間が30分未満**（内部洗浄は最低1時間かかる）

## 📖 ミニ用語辞典（読まなくても大丈夫だけど知っておくと安心）

| 用語 | 意味 |
|------|------|
| **高圧洗浄** | 強い水の勢いで内部の汚れを吹き飛ばす方法 |
| **分解洗浄** | エアコンの部品を外してから洗浄する、より徹底的な方法 |
| **お掃除機能付き** | エアコン自身が自動でフィルターを掃除する機能（付いていると料金が上がる）|
| **防カビコート** | 洗浄後に吹きかけるカビ防止スプレー（無料でやってくれる業者も多い）|
| **損害賠償保険** | 作業中にエアコンが壊れたり、床が濡れたりしたとき、業者が弁償してくれる保険 |
| **明朗会計** | 最初に伝えた料金から変わらない、追加請求しない誠実な料金体系 |

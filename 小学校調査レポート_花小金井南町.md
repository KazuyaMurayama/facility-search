# 小学校調査レポート
## 東京都小平市・花小金井南町からの通学候補校

---

## 結論・総評

花小金井南町（西武新宿線「花小金井駅」徒歩圏）から通いやすい圏内で、**制服・優れたカリキュラム・社会で生きる力・自然体験／自主性**のいずれかを満たす小学校を調査した結果、以下6校を候補として特定した。

### 候補校 早見表

| # | 学校名 | 種別 | 制服 | 花小金井南町からの目安 | 最も合致する条件 |
|---|--------|------|:----:|----------------------|----------------|
| 1 | **サレジオ小学校** | 私立（カトリック） | なし | バス約20分 | 自然体験・共生・少人数 |
| 2 | **東京学芸大学附属小金井小学校** | 国立 | あり※ | 約20〜25分 | 研究教育・宿泊学習・自主性 |
| 3 | **聖徳学園小学校** | 私立 | あり | 約35〜40分 | 思考力・リーダーシップ・英語 |
| 4 | **吉祥寺学園初等部**（旧・武蔵野東小学校） | 私立 | あり | 約30分 | インクルーシブ・共生・ディベート |
| 5 | **明星小学校** | 私立 | あり※ | 約35〜40分 | 探究・体験・英語・一貫校 |
| 6 | **成蹊小学校** | 私立（ユネスコスクール） | あり | 約30〜35分 | 自立・自主性・独自教科・一貫校 |

> ※制服の有無は公式サイト上で明示的に確認できなかった項目を含む。直接確認を推奨。

### 目的別おすすめ

- **とにかく近い・費用を抑えたい** → **サレジオ小学校**
- **国公立で研究型の教育を受けさせたい** → **東京学芸大学附属小金井小学校**
- **考える力・リーダーシップを系統的に育てたい** → **聖徳学園小学校**
- **多様性・共生・思いやりを体感させたい** → **吉祥寺学園初等部**
- **探究・理数・英語の一貫教育** → **明星小学校**
- **自主性・独自教科・精神性の教育** → **成蹊小学校**

### 1. サレジオ小学校

📍 東京都小平市上水南町4-7-1  
🚌 花小金井駅南口より立川バス「国分寺行」→「情報通信研究機構前」または「サレジオ通り」下車 徒歩1〜5分（約20分）

[📍 Googleマップ](https://www.google.com/maps/search/サレジオ小学校+小平市上水南町) ｜ [🌐 公式サイト](https://www.salesio.ac.jp/) ｜ [📖 みんなの小学校情報](https://www.minkou.jp/primary/school/31339/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 私立（カトリック・サレジオ修道会） |
| 制服 | なし（私服。体育着等の指定品あり） |
| 学級規模 | 1学年1クラス・定員22名（児童総数132名） |
| 給食 | なし（弁当持参。業者配達弁当の注文可） |
| 学費 | 私立の中でかなり安め（保護者評価） |
| 創立 | 1947年 |
| 系列校 | サレジオ中学校（小平・男子校）→ サレジオ高専（町田）→ サレジオ学院（横浜） |

#### 教育の特色

聖ヨハネ・ボスコの「予防教育法」に基づく。罰則よりも**信頼関係を基盤**に子どもを育てる思想が根底にある。教師と児童の信頼関係を最優先にした穏やかな教育環境が特徴。

- **少人数制**：1クラス22名で個別対応を徹底。競争・詰め込みとは無縁。
- **自然体験・課外活動**：田植え・稲刈り・夏のキャンプ（2学年合同）・水泳大会など体験行事が充実。
- **道徳（宗教）**：毎朝の聖歌と祈り、月1回のバイブルクラス。ただし宗教を強制しない方針（保護者口コミ）。
- **感性教育**：6年間を通じて様々な楽器を経験する音楽カリキュラム。
- **社会貢献**：老人ホーム訪問などの地域活動。
- **平和学習**：1年生から継続し、6年生の修学旅行で広島を訪問。
- **英語**：3・4年生は週1時間（英語活動）、5・6年生は週2時間（英語科）。

#### 合致する条件

- ✅ 条件3：社会で生きていく力（信頼・共生・平和学習・地域貢献）
- ✅ 条件4：自然体験（田植え・稲刈り・キャンプ）、自主性（少人数による自立的な学び）

### 2. 東京学芸大学附属小金井小学校

📍 東京都小金井市貫井北町4-1-1  
🚌 JR武蔵小金井駅北口より京王バス6系統「中央大学循環」→「学芸小学前」下車（花小金井南町から約20〜25分）

[📍 Googleマップ](https://www.google.com/maps/search/東京学芸大学附属小金井小学校) ｜ [🌐 公式サイト](https://www2.u-gakugei.ac.jp/~kanesyo/) ｜ [📖 受験ガイド](https://shogakko-juken.jp/tokyogakugeidaigakuhuzokukoganei/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 国立（東京学芸大学附属） |
| 制服 | あり（要確認） |
| 学級規模 | 1学年3クラス（児童総数619名） |
| 給食 | あり（校内調理場での手作り給食・週5回） |
| 学費 | 公立小学校相当（給食費年額約6.5万円・教材費年額約9,600円） |
| 創立 | 1911年 |
| 進学 | 卒業生の70〜80%が附属小金井中学校へ進学（学内試験） |

> ⚠️ **国立のため居住地制限あり**。小平市からの受験可否は学校に直接確認が必要。

#### 教育の特色

東京学芸大学の附属校として、最先端の教育研究を実践しながら児童を育てる。全国の教員養成に影響を与える研究授業を公開しており、常に「学びの最前線」を走る環境。

- **研究テーマ**：「理解を深め、知を創造する子の育成 ― 子どもの思考を媒介する『メディア』に着目して」（現在の研究主題）。
- **宿泊学習**：3〜6年生で計6回（千葉県勝浦市・長野県茅野市）。21泊27日に及ぶ体験学習は全国屈指の規模。
- **自主的学び**：子どもが学びを自ら構築する「主体的な学習」を全教科で推進。
- **英語**：1クラス3人体制の手厚い英語教育。
- **給食**：学習内容と連携したメニューを毎回設計（食育との連携）。

#### 合致する条件

- ✅ 条件1：制服あり
- ✅ 条件2：カリキュラムが優れている（全国水準の研究型教育）
- ✅ 条件3：社会で生きていく力（主体的学び・探究）
- ✅ 条件4：自然体験（宿泊学習6回）、自主性（子ども主体の学習設計）

### 3. 聖徳学園小学校

📍 東京都武蔵野市境南町2-11-8  
🚃 JR中央線「武蔵境駅」より徒歩3〜5分（花小金井南町から電車乗り継ぎ約35〜40分）

[📍 Googleマップ](https://www.google.com/maps/search/聖徳学園小学校+武蔵野市境南町) ｜ [🌐 公式サイト](https://el.shotoku.ed.jp/) ｜ [📖 東京私立初等協会](https://www.shigaku.elementary-school.tokyo/schooldata/182/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 私立（聖徳太子の教えを建学の精神に） |
| 制服 | あり |
| 学級規模 | 1学年2クラス・定員34名（児童総数369名） |
| 給食 | なし（弁当持参。仕出し弁当注文可） |
| 預かり | 19:00まで対応 |
| 創立 | 1951年 |
| 系列校 | 聖徳学園幼稚園・聖徳学園中学・高等学校 |

#### 教育の特色

「考える力を一生の財産にする」を掲げた知能教育の名門。**1969年から1年生より教科担任制を導入**（全国唯一とされる）。

- **知能訓練**：1〜4年生対象。ギルフォード博士の理論に基づき90通りの知能因子を刺激する週1時間の独自授業。
- **英語**：1〜2年生は週1時間、3〜4年生は週2時間、5〜6年生は週3時間。ネイティブ教員・少人数制（約15名）。
- **理科実験**：年間150回以上実施。
- **リーダーシップ教育**：スティーブン・コヴィー「7つの習慣」に基づく「リーダー・イン・ミー」プログラムを全校展開。自己管理・主体性・社会性を体系的に育成。
- **特別研究**（5・6年生）：枠にとらわれない自由で大胆な研究学習。
- **林間学校**（3年生以上）：3泊4日の集団宿泊体験。
- **スキー学校・オーストラリア国際交流**も実施。
- **児童自治**：スポーツ大会は企画から運営まで児童が担当。

#### 合致する条件

- ✅ 条件1：制服あり
- ✅ 条件2：カリキュラムが優れている（知能訓練・全国唯一の教科担任制・年150回の理科実験）
- ✅ 条件3：社会で生きていく力（7つの習慣・リーダーシップ教育・児童自治）

### 4. 吉祥寺学園初等部（旧・武蔵野東小学校）

📍 東京都武蔵野市緑町2-1-10  
🚌 西武新宿線「西武柳沢駅」より関東バス約10分（花小金井南町から約30分）

> ⚠️ **2026年4月より「吉祥寺学園初等部」に校名変更**。公式URLも `https://www.k-g.ed.jp/es/` に変更済み。

[📍 Googleマップ](https://www.google.com/maps/search/吉祥寺学園初等部+武蔵野市緑町) ｜ [🌐 公式サイト](https://www.k-g.ed.jp/es/) ｜ [📖 お受験じょうほう](https://www.ojuken.jp/school/musashinohigashi/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 私立（インクルーシブ教育） |
| 制服 | あり（男子：詰め襟/白シャツ、女子：紺/白セーラー） |
| 学級規模 | 普通学級（A・Bクラス）＋ASDクラス（C・D・Eクラス）の混合 |
| 創立 | 60年以上前（インクルーシブ教育として） |

#### 教育の特色

**定型発達児と自閉症スペクトラム（ASD）の子どもが共に学ぶインクルーシブ教育**を60年以上実践する全国でも希少な学校。「一人ひとりはみんな違うのに、みんな一緒に過ごし、大きく成長する」という教育哲学が根底にある。

- **専科制**：音楽・図工・理科は専科の一流指導者が担当。
- **選択授業**（5・6年生）：美術・テニス・陸上・スキー・音楽から選択。
- **東京国立博物館との提携**による特別授業。
- **つながりの時間**：1〜6年の異学年グループ＋普通部・ASDクラス横断の交流活動。
- **ディベート授業**（4年生〜）：論理的思考・伝える力を育成。
- **こころの時間**：思いやり・助け合いを育む道徳教育。

#### 合致する条件

- ✅ 条件1：制服あり
- ✅ 条件3：社会で生きていく力（インクルーシブ教育・多様性・共生・ディベート）
- ✅ 条件4：自主性（異学年交流・選択授業）

### 5. 明星小学校

📍 東京都府中市栄町1-1  
🚌 JR中央線「国分寺駅」より京王バス「府中駅行」約7分→「明星学苑」下車（花小金井南町から約35〜40分）

[📍 Googleマップ](https://www.google.com/maps/search/明星小学校+府中市栄町) ｜ [🌐 公式サイト](https://www.meisei.ac.jp/es/) ｜ [📖 お受験じょうほう](https://www.ojuken.jp/school/meisei/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 私立（明星学苑・一貫校） |
| 制服 | あり（要確認） |
| 系列校 | 明星幼稚園・明星中学・高等学校・明星大学（同一キャンパス） |
| 放課後 | 預かり「メイセイッコ」あり |

#### 教育の特色

「個性を丁寧に見守り好奇心を引き出しながら、多様な学びと本物の体験を通じて『賢さ』と『豊かさ』を育む」を教育方針に掲げる。幼稚園から大学まで同一キャンパスという稀有な一貫教育環境。

- **AA授業**（算数の独自プログラム）：図形・数のゲームやパズルを活用し、毎時間「考える面白さ」を体感。
- **ゼミ探究**（5・6年生）：自分で選んだテーマを少人数で深掘りする探究型学習。
- **英語**：スピーキング中心。イングリッシュキャンプ・英検受験をサポート。
- **理科実験**：低学年から導入。知的好奇心を刺激する授業設計。
- **宿泊学習**：共同生活で協力・規律を身に付ける体験を重視。
- **異学年交流**：明星学苑内の幼稚園〜高校との交流で多世代との関わりを促進。

#### 合致する条件

- ✅ 条件1：制服あり
- ✅ 条件2：カリキュラムが優れている（AA授業・ゼミ探究・探究自走の学び）
- ✅ 条件4：自然体験・体験教育（宿泊学習・理科実験・本物の体験重視）

### 6. 成蹊小学校

📍 東京都武蔵野市吉祥寺北町3-3-1  
🚌 西武新宿線「西武柳沢駅」より関東バス（吉祥寺駅行）→「成蹊学園前」下車 約15分（花小金井南町から約30〜35分）

[📍 Googleマップ](https://www.google.com/maps/search/成蹊小学校+武蔵野市吉祥寺北町) ｜ [🌐 公式サイト](https://www.seikei.ac.jp/elementary/) ｜ [📖 東京私立初等協会](https://www.shigaku.elementary-school.tokyo/schooldata/187/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 私立（成蹊学園・ユネスコスクール加盟） |
| 制服 | あり（白シャツ・ピケ帽、女子はスカーフ付き） |
| 系列校 | 成蹊中・高等学校・成蹊大学（一貫校） |

#### 教育の特色

「自立・連帯・創造」を教育目標に掲げるユネスコスクール加盟校。**「こみち科」「桃李科」という2つの独自教科**を持ち、他の私立小学校とは一線を画す教育哲学が特徴。

- **こみち科**（独自教科）：地域から地球規模の課題を探究・表現する体験活動。1年生から6年生まで継続。
- **桃李科**（独自の道徳）：「尊い心」に気づく感性教育。心力歌（1913年制作・全8章）の音読を毎日実施。
- **凝念（ぎょうねん）**：創立以来続く朝の精神集中法（黙想）。授業の始めに心を整える習慣。
- **桃の会**（児童自治組織）：学級桃の会と5・6年生の13委員会で構成。「自ら発見・解決する力」を育成。
- **夏の学校**：仲間と共に感性豊かにやり遂げる宿泊体験学習。
- **卒業研究**（6年生）：深い探究活動の集大成。
- **日記指導**：全学年で継続。教師と児童の心の対話、考える習慣、伝える力を同時に育む。
- **英語・国際学級交流**：5・6年生でオーストラリア体験学習を実施。
- **専科制**：1年生から音楽・美術・体育・英語・読書で専科を導入。

#### 合致する条件

- ✅ 条件1：制服あり
- ✅ 条件2：カリキュラムが優れている（独自教科・専科制・ユネスコスクール）
- ✅ 条件3：社会で生きていく力（こみち科・桃李科・児童自治・日記指導）
- ✅ 条件4：自然体験（夏の学校・卒業研究）、自主性（桃の会・自学自修）

4. アクセス時間はバス・電車の乗り継ぎ目安。スクールバスの有無・自転車通学可否も各校に確認のこと。

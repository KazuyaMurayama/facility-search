## 0. はじめに：ビタミンD検査の基礎知識

「25(OH)D（25-ヒドロキシビタミンD）」は、血液中のビタミンD貯蔵量を反映する指標で、体内のビタミンDが充足しているかを最も正確に評価できる検査です。

### 保険適用 vs 自由診療

| 区分 | 条件 | 自己負担の目安 |
|---|---|---|
| **保険適用** | 骨粗鬆症・くる病・骨軟化症の診断時・治療中など、医師が必要と判断した場合のみ（診断時1回、その後3か月に1回が上限） | 3割負担で **約500〜900円**＋診察料 |
| **自由診療（自費）** | 予防・セルフチェック目的（=今回の主目的） | **約1,500〜10,000円**（施設で差が大きい） |

> ⚠️ **重要**：「足りているか調べたい」という予防目的の場合、ほとんどのケースで**自費診療**になります。骨粗鬆症などの診断がない限り保険は使えません。

### 判定の目安（25(OH)D 血中濃度）

| 数値 | 判定 |
|---|---|
| 20 ng/mL 未満 | ビタミンD **欠乏** |
| 20〜30 ng/mL 未満 | ビタミンD **不足** |
| 30 ng/mL 以上 | **充足**（国内基準） |
| 40〜60 ng/mL | アレルギー・免疫・骨対策で理想とされる層 |

## 3. 【セルフチェック】ビタミンD欠乏 リスク＆症状チェックリスト

> ⚠️ これは医療診断ではありません。当てはまる項目が多いほど不足の可能性が高く、検査・受診を検討する目安です。

### A. 生活習慣・体質リスク（日光と摂取）
- [ ] 日中はほとんど室内で過ごす（在宅勤務・デスクワーク中心）
- [ ] 外出時は日焼け止め・日傘・長袖で紫外線をしっかり防いでいる
- [ ] 日照の少ない季節（冬）や曇天・北側中心の生活
- [ ] 魚（特に鮭・サンマ・イワシなど）をあまり食べない
- [ ] きのこ類・卵黄・乳製品の摂取が少ない
- [ ] ダイエット・偏食・菜食中心で脂質をあまり摂らない
- [ ] BMIが高め（肥満傾向）／または過度に痩せている
- [ ] 高齢である（加齢で皮膚での合成が低下）
- [ ] 肌の色が濃いめ（メラニンが多いと合成効率が下がる）

### B. 体調・症状サイン
- [ ] 慢性的に疲れやすい・だるさが抜けない
- [ ] 骨や関節がなんとなく痛む・重い
- [ ] 筋力低下を感じる／つまずき・転びやすい
- [ ] 風邪や感染症にかかりやすい・治りにくい
- [ ] 気分が落ち込みやすい（特に冬季）
- [ ] 髪が抜けやすい・薄くなってきた
- [ ] 傷の治りが遅い
- [ ] 筋肉のけいれん・こむら返りが起きやすい

### C. 該当する人は特に検査推奨（ハイリスク群）
- [ ] 骨粗鬆症・骨折歴がある
- [ ] 妊娠中・授乳中である
- [ ] アレルギー・自己免疫疾患がある
- [ ] 腸の病気や脂肪吸収に問題がある（クローン病・胃腸切除など）

### 判定の目安
| チェック数（A+B合計） | 目安 |
|---|---|
| 0〜2個 | 不足リスクは低め。バランス維持を |
| 3〜5個 | やや不足の可能性。食事・日光＋検査検討 |
| 6個以上 | 不足の可能性が高い。**検査を強くおすすめ** |
| Cが1つでも該当 | 個数に関わらず医療機関で相談を |

### 不足を感じたら（生活面のセルフケア）
- 晴れた日に**1日15〜30分程度**、手や顔に日光を浴びる（季節・地域で必要時間は変動）
- **鮭・サンマ・イワシ・しらす**などの魚、**きのこ（特に天日干し）**、卵黄を意識的に摂る
- 必要に応じてビタミンDサプリメント（過剰摂取に注意。検査値を見て医師・薬剤師と調整）

## 4. 受診前チェック（実行アクション）

1. **目的を決める**：保険で安く（症状あり）→ ③／予防で総合把握 → ①②／手軽に数値だけ → ④
2. **電話で確認**：「25(OH)ビタミンD（25OHD）の血中濃度を**自費でも測れますか？費用は？**」と聞く
3. **空腹要否・予約要否・結果が出るまでの日数**を確認（ビタミンDは通常2〜4日〜数週間）
4. セルフチェックでC群に該当する場合は、医師に必ずその旨を伝える
